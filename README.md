# Image Denoising using Deep Learning

## Overview

This project explores deep learning techniques for image denoising using PyTorch.

The goal is to recover clean images from noisy observations while preserving structural details.

---

## Current Model

- Baseline CNN
- MSE Loss
- Adam Optimizer
